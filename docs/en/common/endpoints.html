Endpoint Summary

The Create Order, Query, and Refund functions for the vast majority of channels all use the same three endpoints, distinguished only by the service parameter for the specific payment channel.

FunctionEndpoint PathMethodDescription
Create Order/payPOSTCreate a payment order
Invoke Payment/pay-againPOSTRe-obtain payment link (some channels)
Order Query/queryPOSTQuery order status
Refund/refundPOSTInitiate refund
Refund Query/refundQueryPOSTQuery refund status
