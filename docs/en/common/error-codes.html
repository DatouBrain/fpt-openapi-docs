Error Codes

Doc Version: V1.0

This page lists all error codes that may be returned by the Fintech Payment OpenAPI, divided into common error codes and card payment specific error codes.

Error Response Structure

When an API call fails, the following JSON structure is returned:

JSON{
  "message": "錯誤描述信息",
  "errors": [
    {
      "field": "出錯字段名",
      "message": "字段錯誤描述",
      "code": "錯誤碼"
    }
  ]
}

A message other than SUCCESS indicates an API call error.
