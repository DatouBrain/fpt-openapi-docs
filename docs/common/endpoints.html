ℹ️ 基礎 URL

所有 API 請求均需以以下基礎 URL 為前綴，再加上各接口特定的路徑。

環境地址

環境基礎 URL
UAT（測試環境）https://uat.fintechpaymenthub.com/openapi/v1/
Production（生產環境）https://api.fintechpaymenthub.com/openapi/v1/

端點摘要

絕大多數渠道的下單、查詢、退款功能都使用相同的三個端點，僅透過 service 參數區分具體支付渠道。

功能端點路徑請求方法說明
下單/payPOST創建支付訂單
調起支付/pay-againPOST重新獲取支付鏈接（部分渠道支持）
訂單查詢/queryPOST查詢訂單狀態
退款/refundPOST發起退款
退款查詢/refundQueryPOST查詢退款狀態
