數據格式

文檔版本：V1.0

本頁面說明 Fintech Payment OpenAPI 的數據傳輸格式，包括請求格式、響應格式及公共參數。

請求格式

API 支持兩種請求提交方式：

標準模式（JSON POST）

絕大多數支付渠道採用 HTTPS POST + JSON 格式提交數據：

  • 請求方法：POST
  • Content-Type：application/json
  • 字符編碼：UTF-8
  • 數據必須簽名
JSON{
  "service": "pay.weixin.web.intl",
  "order_no": "00f78e479fe74a11a73be0feb9999749",
  "body": "測試支付",
  "amount": 1.00,
  "callback_url": "https://your-domain.com/result",
  "notify_url": "https://your-domain.com/notify",
  "client_ip": "127.0.0.1",
  "client_id": "1002",
  "store_id": "1001002",
  "nonce_str": "ptf3tyjv",
  "sign": "64950ED7B13489114C671EFD24C45AC3",
  "lang": "en_US"
}

託管模式（FORM POST）

Global Payments Online 託管模式使用 HTML 表單提交，適合直接在瀏覽器中跳轉到支付頁面：

  • 請求方法：POST
  • Content-Type：application/x-www-form-urlencoded
  • 字符編碼：UTF-8
  • 數據必須簽名
HTML<form action="https://uat.fintechpaymenthub.com/openapi/v1/pay" method="post">
  <input type="hidden" name="service" value="pay.globalpayment.card.notpresent">
  <input type="hidden" name="client_id" value="1001">
  <input type="hidden" name="store_id" value="1001001">
  <input type="hidden" name="order_no" value="ORDER001">
  <input type="hidden" name="amount" value="100">
  <input type="hidden" name="sign" value="XXXXX">
</form>

響應格式

所有接口返回 JSON 格式數據。根據 message 字段判斷請求是否成功：

成功返回

JSON{
  "message": "SUCCESS",
  "order_no": "00f78e479fe74a11a73be0feb9999749",
  "transaction_id": "FTP202012181649370001",
  "submit_time": "20201218164937",
  "nonce_str": "o66w9pft",
  "sign": "3F6C05280C90E5BB1330524FD61C6121"
}

message 返回 SUCCESS 表示接口調用成功。返回結果中包含 sign 字段，商戶應驗證簽名以確保數據完整性。

錯誤返回

JSON{
  "message": "Missing Merchant ID",
  "errors": [
    {
      "field": "mch_id",
      "message": "Missing Merchant ID",
      "code": "MISSING_FIELD"
    }
  ]
}

message 返回非 SUCCESS 時表示接口調用出錯。errors 數組包含具體的錯誤字段和錯誤碼。

公共請求參數

以下參數在所有接口中通用：

參數名類型必填說明是否參與簽名
serviceString(40)必填支付服務標識，區分不同支付渠道✅
client_idString(32)必填商戶客戶端 ID✅
store_idString(50)必填門店 ID✅
order_noString(32)必填商戶系統內部的訂單號，需確保唯一✅
nonce_strString(32)必填隨機字符串，防重放攻擊✅
signString(64)必填數字簽名，詳見數字簽名❌
langString(10)選填語言設置，如 en_US、zh_CN❌
