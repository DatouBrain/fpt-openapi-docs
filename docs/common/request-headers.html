請求頭說明

文檔版本：V1.0

本頁面說明調用 Fintech Payment OpenAPI 時所需的 HTTP 請求頭。

標準請求頭

使用 JSON POST 方式調用接口時，請設置以下請求頭：

請求頭值必填說明
Content-Typeapplication/json必填請求體為 JSON 格式
Acceptapplication/json建議響應體為 JSON 格式
Accept-Charsetutf-8建議字符編碼為 UTF-8

託管模式請求頭

Global Payments Online 託管模式使用 HTML 表單提交，請求頭有所不同：

請求頭值說明
Content-Typeapplication/x-www-form-urlencoded表單提交格式
ℹ️ 字符編碼

所有請求和響應均使用 UTF-8 編碼。簽名計算時也必須使用 UTF-8 編碼將字符串轉換為位元組流。
