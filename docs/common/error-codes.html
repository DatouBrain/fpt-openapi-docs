錯誤碼說明

文檔版本：V1.0

本頁面列出 Fintech Payment OpenAPI 可能返回的所有錯誤碼，分為通用錯誤碼和卡支付專屬錯誤碼。

錯誤響應結構

當接口調用出錯時，返回以下 JSON 結構：

JSON{
  "message": "錯誤描述信息",
  "errors": [
    {
      "field": "出錯字段名",
      "message": "字段錯誤描述",
      "code": "錯誤碼"
    }
  ]
}

message 返回非 SUCCESS 時表示接口調用出錯。
